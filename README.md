##  Objective
The goal of this project is to build the **memory system** of an AI Research Assistant.
Designed a **semantic search engine** that retrieves academic documents based on
meaning rather than keywords.

This phase focuses on:
- Document embeddings
- Vector databases
- Semantic retrieval
- Retrieval quality analysis

---

### Semantic Retrieval
Application provides:
- A query input box
- A configurable `top-k` value
- Clearly ordered retrieval results based on relevance

### Retrieval Evaluation & Analysis

- Multiple queries
